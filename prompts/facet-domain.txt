of emojis, each with its official name and CLDR keywords.

FACET: domain

TASK
For each emoji, assign zero or more tags from the vocabulary above.

association is one most people would recognize immediately.

RULES
1. Never invent a tag outside the vocabulary. If nothing fits, return an

   beats recall, humans review everything below 0.8.
6. Output only valid JSON, no commentary, no markdown fences.
